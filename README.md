## XAR Migration
A Solidity smart contract to facilitate the migration of XAR tokens to AVAIL tokens.

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test -vvv
```

### Deploy

```shell
$ forge script script/XARMigration.s.sol --rpc-url <your_rpc_url> --private-key <your_private_key>
```
